Practica de CSS

Para aplicarlo tenemos tres opciones:

1. En linea Código embebido

2.-Códig Interno -Código en la cabecera
El código debe ir en la cabecera del file
style

background:blue;
}

Hipervinculos

Hipervinculos: Abrir en nueva pestaña

Atributo target="_blank": Este atributo se utiliza para especificar cómo se debe abrir el hipervínculo. Puede ser_blank,_self_parent, o cualquier nombre diferente que no sea de los cuatro mencionados, que se considera_blank para abrir en una nueva pestaña o ventana, Abrir en una nueva pestaña


Hipervinculos: Abrir en la misma pestaña

Para abrir un hipervínculo en la misma pestaña, se debe utilizar el atributo target="_self" en la etiqueta del hipervínculo. Este atributo indica al navegador que el enlace debe abrirse en la misma pestaña donde se encuentra el hipervínculo. Por ejemplo: Texto. Sin embargo, esta técnica no es compatible con todas las versiones de Power BI, y puede requerir la inclusión de ideas similares en la plataforma para su implementación. Abrir en una nueva pestaña


Hipervinculos: Abrir en pestaña, modo seguro

Atributo rel="noopener noreferrer": Este es el uso combinado de los atributos noopener y noreferrer, que se utilizan para mejorar la seguridad y la privacidad al abrir hipervinculos. Al usar estos atributos, se puede garantizar que los hipervinculos se abran en una nueva pestaña o ventana, proporcionando una experiencia de navegación más segura y responsable. Abrir en una nueva pestaña (Modo seguro)
